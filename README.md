# Todo List

This is an app that helps you organize your todos with which you can add, complete and delete them.

## Screenshot

![Screenshot](./src/images/Screenshot.png)

To use this app run these commands:

- ### `npm install`
- ### `npm start`

## Built with

- CSS custom properties
- Normalize.css
- React.js
